# My AI Todo List

> **⚠️ Experiment / 实验作品**
> 
> This is a desktop application created by a developer with **zero coding background**, entirely through dialogue with AI.
> 这是一个由**零代码基础**的开发者，完全通过与 AI 对话构建的桌面端应用。
> 
> The purpose of this repository is to test AI's capability in building software from scratch.
> 本仓库旨在测试 AI 从零构建软件的能力，仅供展示和娱乐。
